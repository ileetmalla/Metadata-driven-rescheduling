
## What is included

- Per-run `cluster_power.csv` files used for cluster-power box plots.
- Per-run `service_state.csv` files used for threshold-state and profile-distribution analysis.
- Per-run `experiment_run.csv` metadata.
- Per-run `reschedule_events.csv` files for controller movement/action analysis.
- Per-run `power_summary_*.json` and `runtime_state_*.json` summaries.
- Intent configuration YAML files found in the bundles.
